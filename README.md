# Syarat menggunakan software ini
Setelah era windows 7 berakhir, software ini mendapat tantangan ketika di th 2020 terdapat upgrade yg mem-blok penggunaan flash dan berakibat banyak windows 8 keatas yang tidak bisa menjalankan, jadi syaratnya :

1. Windows 7 atau versi lain yang tidak banyak update yg mem-blok flash player (googling aja)
2. Instal aplikasi multi messenger pidgin (www.pidgim.im), dan install plugin seperti telegram, whatsapp dll
3. Unblok anti virus yang menyebabkan macro tidak jalan
4. Pastikan anda sudah bisa transaksi ke server pulsa anda lewat jabber, wa maupun telegram
